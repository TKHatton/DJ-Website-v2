03 / Beginner agent course

Three Gears.

A three-session live course built for working women (and anyone else) who already use Claude or ChatGPT and want to take the next step. We move from "I use AI" to "I built an agent that runs without me." Three sessions. Real outputs. No code required, but you'll see what's underneath.

  • Session 1. The First Gear: scoping a useful agent.
  • Session 2. The Second Gear: building it without coding.
  • Session 3. The Third Gear: making it run on its own.
  • Two-page handout. Reference material for after.
  • Live group, small cohorts. 10 to 20 attendees per session.
  • Recordings included if you miss a session.

04 / MAOS Masterclass

Building agentic systems
that actually work.

An advanced 90 to 120 minute session for builders, operators, and technical leaders who want to design real agentic systems. Five modules, five outcomes. The session walks through capability-scoped agent design, the reasoning behind it, and the failure modes of the "forty single-task agents" approach.

  • Module 1. Why agent design fails most teams.
  • Module 2. Capability scoping vs task scoping.
  • Module 3. The five-to-eight agent operating system.
  • Module 4. Memory, observability, and recovery.
  • Module 5. Shipping it without watching it.
  • Takeaway kit. Seven artifacts you keep.
